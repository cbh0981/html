대경이의 HTML

  1. <!doctype html>: 이것은 html로 되어있는 문서이다.(일종의그냥 약속이다)
  2. <html>: html의 시작과 끝(이것도 하나의 약속)
  3. <head>: 제목이되는부분을 내용부분과 나눠주는태그 중요한 키워드를 나타내는 문단을 나눌때 쓰는 태그 (이것도하나의약속?)
  4. <title>:웹페이지를 설명해주는 태그 탭에 설명을 나타냄
  5. <meta charset="?"> : char:문자 set:표시 ?이라는 형식으로 문자를 표현해라라는태그
  6. <body>: 내용적인부분을 나눠주는 태그(이것도하나의 약속)
  7. <h1,h2,h3,h4,h5,h6>:숫자의따라 제목또는 키워드의 중요도를 나타냄 1이 제일 크기가크다
  8. <ol,ul>:oredred list, unordered list 의약자 ol은 숫자로 리스트를 나타낼수있다 ul은 점으로 나타낼수있다
  9. <a href="사이트주소"> target:_blank (새창에서열라는의미이다) title:(설명을나타낼수있다);:anchor의 약자 약간 시적인표현 태그로 문자를감싸면 문자를 클릭하여 링크로들어갈수있다
  10. <strong>:문자를 태그로감싸면 문자가 진하게강조되어표현된다
  11. <u>:문자를 태그로감싸면 문자에 밑줄이생긴다
  12. <img src=이미지파일명 width=(그림의크기설정가능)>:해당웹에 이미지를 띄울수있다
